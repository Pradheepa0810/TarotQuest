# Tarot Quest

This project is now organized into modular static assets for easier maintenance.

## Structure

- `tarot-quest.html` - page markup and script/style references only
- `deck-data.js` - tarot deck/card data source
- `whimsical cottage.jpeg` - local logo/home image
- `assets/css/main.css` - all app styles
- `assets/js/app.js` - app logic (state, routing, rendering, interactions)

## Run

Open `tarot-quest.html` in a browser.

## Notes

- Keep data definitions in `deck-data.js`.
- Keep visual changes in `assets/css/main.css`.
- Keep behavior and UI logic in `assets/js/app.js`.
